# Video Looper (for practicing listening)
동영상과 자막 파일을 자막 기준으로 편리하게 반복 재생하며 영어(다른 언어) 듣기 연습을 할 수 있는 웹앱

## Features
- 로컬 디스크에 있는 동영상과 자막 파일 선택
- 자막파일은 SRT 포맷 지원
- 자막 단위별로 선택하여 반복 재생
- 반복 회수 지정
- 전체 자막 순차 재생 / 현재 자막만 재생
- 동영상과 자막에 따라 여러 언어에 사용 가능
